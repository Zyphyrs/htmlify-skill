What is last30days?

last30days is a Claude Code skill that researches any topic across the last 30 days of internet activity. A single command sweeps Reddit threads, X/Twitter posts, YouTube videos, TikTok content, Hacker News discussions, Polymarket prediction markets, Instagram, and the open web — then synthesizes a briefing.

Zero mandatory API keys for the base run. Unlock deeper coverage layer by layer as you add optional keys. Briefings are auto-saved so you can build a research archive over time.

Install via add-skill
One-liner install

Requires Claude Code with the add-skill helper configured. The skill is added to your active Claude Code session.

Auto Setup

Run the built-in setup wizard to configure sources interactively:

The wizard walks through each source, checks what's already configured, and prompts for missing keys only.

Manual .env Config
All supported keys
# Required for nothing — all optional SCRAPECREATORS_API_KEY=sc_... # Reddit comments, TikTok, Instagram TWITTER_BEARER_TOKEN=AAAA... # X/Twitter posts (or use --x-handle) FIRECRAWL_API_KEY=fc-... # Deep URL scraping YOUTUBE_API_KEY=AIza... # YouTube search (ScrapeCreators fallback) POLYMARKET_API_KEY= # Public API, key optional # Output directory (default: ~/Documents/Last30Days/) LAST30DAYS_OUTPUT_DIR=~/Documents/Last30Days/
ScrapeCreators

ScrapeCreators is the recommended data layer for social platforms. The free tier gives you 100 free API calls which unlocks Reddit posts and YouTube in the base run. Paid tier unlocks Reddit comments, TikTok, and Instagram.

  • ✓ Sign up at scrapecreators.com — no credit card for free tier
  • ✓ Copy your API key and add to SCRAPECREATORS_API_KEY
  • ✓ Free tier resets monthly
Command Examples
# Basic research run /last30days "vibe coding trend" # Quick mode — fewer sources, faster output /last30days "AI agents" --quick # Deep dive — all sources, longer context /last30days "Claude Code" --deep # Custom date window (default is 30 days) /last30days "llm benchmarks" --days=7 # Run as background agent (non-blocking) /last30days "crypto market sentiment" --agent # Include X/Twitter from a specific handle's perspective /last30days "indie hacking" --x-handle=levelsio # Deep + save to custom path /last30days "open source LLMs" --deep --days=14
Output Format

Each run produces a structured markdown briefing saved to ~/Documents/Last30Days/. The briefing contains:

  • → Executive summary (3–5 bullet points)
  • → Per-source signal summary with top posts/links
  • → Cross-platform convergence highlights
  • → Sentiment polarity per platform
  • → Prediction market probability if Polymarket has relevant markets
  • → Raw source metadata for follow-up research
Flags Reference
--quick Fast mode — web + HN + Polymarket only. Returns in ~30 seconds. --deep All configured sources, extended context window per source, longer synthesis. --days=N Override the default 30-day window. Use 7 for week-in-review, 90 for quarterly. --agent Run as a background subagent. Claude Code continues responding while research runs. --x-handle= Pass a Twitter/X handle to focus X research through that account's network and replies. --no-save Skip auto-save. Output is printed inline only, nothing written to disk.
